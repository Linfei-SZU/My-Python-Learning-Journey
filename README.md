# My-Python-Learning-Journey

欢迎来到我的 Python 学习之旅仓库！这里将记录我在深圳大学学习 Python 编程过程中的代码、项目和心得体会。

## 🎯 项目简介

这个仓库是我在大学期间学习 Python 编程的成长记录册。它不仅是代码的存储地，更是我个人技能发展的可视化证明。目前仓库包含了两个结合数据分析与条件判断的实用程序。

## ✨ 已实现项目

### 1. 大学生游戏时长分析器
- **文件**: `沉迷游戏判断.py`
- **功能**: 该程序通过分析学生每日的游戏时长，结合其学业成绩，对学生的游戏行为健康度进行评估与提示。它旨在探讨娱乐与学业之间的平衡关系。
- **技术亮点**: 使用条件判断链 (`if-elif-else`) 来实现多层级的状态评估。

### 2. 城市空气质量判定器
- **文件**: `空气质量检测.py`
- **功能**: 根据原始的PM2.5、AQI等关键污染物浓度数据，程序会自动判定当前的空气质量等级（如优、良、轻度污染等），并提供相应的健康出行建议。
- **技术亮点**: 将复杂的空气质量国家标准封装为清晰的程序逻辑，展示了将现实规则转化为代码的能力。

## 🛠️ 技术栈

- **编程语言**: Python 3

## 🚀 如何运行

### 先决条件
你的电脑需要安装有 Python 3.6 或更高版本。

### 运行步骤
1.  在本仓库页面，找到你想要运行的程序文件（如 `沉迷游戏判断.py`）。
2.  点击文件名称进入详情页，然后点击右上角的 **"Raw"** 按钮，将完整的代码复制下来，在你电脑上创建一个同名文件并保存。
3.  打开命令行（终端/命令提示符），导航到该文件所在的目录。
4.  输入以下命令来运行程序：
    ```bash
    python 沉迷游戏判断.py
    ```
    或者
    ```bash
    python 空气质量检测.py
    ```

## 学习心得与规划

### 一、深度复盘：任务完成过程中的挑战与突破

本次创建GitHub仓库并完成首次项目提交的任务，对我而言是一次完整的技术工作流启蒙。整个过程虽然遇到不少困难，但每个困难的解决都带来了宝贵的经验。

起步阶段最大的障碍是网络环境。GitHub的直接访问体验很不稳定，页面加载缓慢，关键操作时常失败。这个问题如果无法解决，后续所有工作都无从谈起。通过搜索和咨询，我了解到Watt Toolkit这款开源加速工具，它专门为开发者访问GitHub、Steam等国际平台提供网络优化。成功配置并使用后，访问体验得到质的提升。这个过程让我深刻认识到，在当今的开发环境中，解决网络、环境配置等“非代码”问题的能力，与编程能力本身同等重要。一个开发者不仅要会写代码，更要善于利用各种工具为自己创造高效的工作环境。

另一个知识盲区是Markdown语法和README文档的规范编写。在此之前，我对如何在网页上优雅地排版一无所知。通过AI助手的指导，我快速理解了Markdown的基本逻辑：用简单的符号实现标题、列表、代码块等元素的格式化。我意识到，在现代技术学习中，遇到不熟悉的知识点时，不必系统性地从头学起，而是应该优先采用“目标驱动”的学习方式——先掌握完成当前任务所需的最小知识集合，快速应用，在实践中加深理解。这种高效获取并应用知识的能力，其价值不亚于知识本身。

### 二、核心启发：GitHub——程序员的动态能力展示平台

浏览了GitHub上众多优秀的Python项目后，再回顾我自己完成任务的整个过程，我对于如何管理个人项目和展示个人技能有了全新的认知。

最核心的启发是：**GitHub就是程序员的“动态作品集”**。对于设计师，作品集是设计稿和效果图；对于程序员，作品集就是由一个个代码仓库构成的技术履历。一个整洁、规范、文档齐全的GitHub主页，能向潜在的导师、合作伙伴或招聘方清晰传递出许多积极信号：严谨的逻辑思维、良好的文档习惯、项目管理的初步意识以及对开源协作文化的认同。

我的`My-Python-Learning-Journey`仓库，就是这个作品集的起点。它不再是一个孤立的作业，而是我个人技术品牌的基石。通过它，我展示的不仅仅是两个用Python实现的功能性程序，更展示了我将现实问题（游戏时长管理、空气质量判断）抽象为计算机逻辑的能力，以及我初步掌握版本控制这一现代软件开发基石工具的意愿。README文件就是这个基石的“门面”，其质量直接决定了访客对我专业度的第一印象。

### 三、未来蓝图：构建持续成长的技术知识体系

基于以上认知，我计划将这个仓库从一个单纯的作业提交地，全面升级为我个人学习与成长的“数字花园”，并制定清晰的执行路径。

**2. 知识管理与开源分享**

**3. 与职业路径深度融合**
